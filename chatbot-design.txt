●Components
1. Frontend(Streamlit)
    - User Authentication
    - Chat Interface for Prompts
2. Authentication Layer(auth.py)
    -Handles user registration
    -JWT token-based session management
3. Project Manager(project-manager.py)
    - Saves & Updates User's Projects
4. LLM Handler(llm-handle.py)
    - Sends Prompts to OpenAI API
    - Returns response based on project types
5. Database(database.py)
    - SQLite structure for users & projects
6. Config(config.py)
    - Stores the env variables
●Data Flow

User logs in/registers → JWT stored → session saved.

User sends a prompt in chat.

llm_handler.py calls OpenAI API → AI generates response/project idea.

Response is saved via project_manager.py → stored in SQLite.

Projects displayed in sidebar history.
